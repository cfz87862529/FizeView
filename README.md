# FizeView
支持各种引擎的视图库

FizeView 没有再新建一个视图语法，它认为没有这个必要！

世面已经有那么多性能不错，满足各类需求的视图框架了，为什么我们还需要 FizeView ？

因为 FizeView 允许你使用任意的视图引擎，但是又使用相同的视图接口，这样的好处很明显：你可以随意选择自己擅长的视图引擎来编写前端代码。

FizeView 有非常完善的参考文档，且其功能追求简洁明了，相信您会喜欢上这样的视图库。
